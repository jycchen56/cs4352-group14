## External Dependencies

The following external dependencies are used in this project. All are managed via `npm`.

### Core Framework
*   **React**
    *   **URL:** [https://react.dev/](https://react.dev/)
    *   **Function:** A JavaScript library for building user interfaces.
*   **React Native**
    *   **URL:** [https://reactnative.dev/](https://reactnative.dev/)
    *   **Function:** A framework for building native apps using React.
*   **Expo**
    *   **URL:** [https://expo.dev/](https://expo.dev/)
    *   **Function:** An open-source platform for making universal native apps for Android, iOS, and the web with JavaScript and React.

### Navigation & Routing
*   **Expo Router**
    *   **URL:** [https://docs.expo.dev/router/introduction/](https://docs.expo.dev/router/introduction/)
    *   **Function:** File-based routing for React Native and web, built on top of React Navigation.
*   **React Navigation** (including `@react-navigation/native`, `@react-navigation/bottom-tabs`, etc.)
    *   **URL:** [https://reactnavigation.org/](https://reactnavigation.org/)
    *   **Function:** Routing and navigation for Expo and React Native apps.
*   **React Router DOM**
    *   **URL:** [https://reactrouter.com/en/main](https://reactrouter.com/en/main)
    *   **Function:** DOM bindings for React Router (likely used for web-specific fallback or compatibility).

### UI & Icons
*   **@expo/vector-icons**
    *   **URL:** [https://icons.expo.fyi/](https://icons.expo.fyi/)
    *   **Function:** Provides icon sets (Ionicons, FontAwesome, etc.) for the UI.
*   **Expo Image**
    *   **URL:** [https://docs.expo.dev/versions/latest/sdk/image/](https://docs.expo.dev/versions/latest/sdk/image/)
    *   **Function:** A cross-platform image component designed for performance.
*   **Expo Font**
    *   **URL:** [https://docs.expo.dev/versions/latest/sdk/font/](https://docs.expo.dev/versions/latest/sdk/font/)
    *   **Function:** Allows loading custom fonts from the web or local assets.
*   **React Native Safe Area Context**
    *   **URL:** [https://github.com/th3rdwave/react-native-safe-area-context](https://github.com/th3rdwave/react-native-safe-area-context)
    *   **Function:** A flexible way to handle safe area insets (notches, home bars) on mobile devices.

### Gestures & Animations
*   **React Native Reanimated**
    *   **URL:** [https://docs.swmansion.com/react-native-reanimated/](https://docs.swmansion.com/react-native-reanimated/)
    *   **Function:** A powerful animation library for React Native.
*   **React Native Gesture Handler**
    *   **URL:** [https://docs.swmansion.com/react-native-gesture-handler/](https://docs.swmansion.com/react-native-gesture-handler/)
    *   **Function:** Declarative API for handling touches and gestures.

### Utilities
*   **Expo Constants**
    *   **URL:** [https://docs.expo.dev/versions/latest/sdk/constants/](https://docs.expo.dev/versions/latest/sdk/constants/)
    *   **Function:** Provides system information about the device and environment.
*   **Expo Haptics**
    *   **URL:** [https://docs.expo.dev/versions/latest/sdk/haptics/](https://docs.expo.dev/versions/latest/sdk/haptics/)
    *   **Function:** Provides haptic feedback on iOS and Android.
*   **Expo Linking**
    *   **URL:** [https://docs.expo.dev/versions/latest/sdk/linking/](https://docs.expo.dev/versions/latest/sdk/linking/)
    *   **Function:** Handles deep links into the application.
*   **Expo Web Browser**
    *   **URL:** [https://docs.expo.dev/versions/latest/sdk/web-browser/](https://docs.expo.dev/versions/latest/sdk/web-browser/)
    *   **Function:** Provides access to the system's web browser.
